# LEMA Capital - Calculadora de Inversión

Calculadora interactiva que permite a los clientes simular su inversión usando los rendimientos históricos reales de los portafolios de LEMA Capital.

## Características

- Simulación con inversión inicial personalizable
- Aportaciones mensuales configurables
- Comparación con benchmarks (IPC México, S&P 500, GBMF2)
- Gráficos interactivos de evolución
- Datos históricos reales de los 4 portafolios
- Responsive (funciona en móvil y desktop)

## Deploy en Vercel

1. Sube esta carpeta a GitHub
2. Conecta el repo con Vercel
3. Deploy automático

## Desarrollo Local

```bash
npm install
npm run dev
```
